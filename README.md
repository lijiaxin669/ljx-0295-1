# 拜年贺卡工坊 🧧

一个基于 Svelte + Canvas 的拜年贺卡制作工具，支持手写祝福、贴纸装饰、笔迹动画和视频导出。

## ✨ 功能特性

- 🎨 **双模板选择**：红金国风 / 简约现代
- ✍️ **Canvas 手写**：支持鼠标/触摸书写，可调节笔刷颜色、粗细、透明度
- 📝 **预制祝福语**：20+ 条经典/现代/幽默/商务祝福语一键插入
- 🏮 **贴纸系统**：灯笼、福字、生肖等 12 款贴纸，支持拖拽、缩放、旋转
- 🎬 **笔迹动画**：逐字显现动画，模拟真人书写效果
- 📹 **视频导出**：支持 15 秒 MP4 / GIF 导出，可分享到微信群
- 💾 **数据持久化**：笔迹数据保存为 JSON，支持二次编辑
- ⚡ **客户端转码**：基于 ffmpeg.wasm，无需服务端

## 🛠️ 技术栈

- **前端框架**：Svelte 4.x + TypeScript 5.x
- **构建工具**：Vite 5.x
- **样式方案**：SCSS + CSS Variables
- **核心渲染**：Canvas 2D API
- **动画驱动**：requestAnimationFrame
- **视频转码**：ffmpeg.wasm v0.12.x
- **状态管理**：Svelte Stores
- **部署方案**：Docker + Nginx

## 🚀 快速开始

### 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器 (http://localhost:5173)
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

### Docker 部署

```bash
# 仅启动静态站点
docker-compose up -d

# 启动静态站点 + 转码 sidecar (可选)
docker-compose --profile transcoder up -d

# 停止服务
docker-compose down
```

访问 http://localhost:8080

## 🔧 Chrome 导出验收步骤

### 前置检查

1. **Chrome 版本**：确保 Chrome 版本 >= 92（推荐 110+）
2. **内存检查**：导出 1080p 视频建议至少 4GB 可用内存
3. **缓存清理**：首次使用建议清理浏览器缓存

### 步骤 1：启动应用

```bash
npm run dev
```

在 Chrome 中打开 http://localhost:5173

### 步骤 2：验证 COOP/COEP 头

1. 按 F12 打开开发者工具
2. 切换到 Network（网络）面板
3. 刷新页面，点击第一个请求（localhost:5173）
4. 在 Response Headers（响应头）中确认：
   ```
   Cross-Origin-Opener-Policy: same-origin
   Cross-Origin-Embedder-Policy: require-corp
   ```

### 步骤 3：验证 SharedArrayBuffer 支持

1. 在 Console（控制台）中输入：
   ```javascript
   typeof SharedArrayBuffer
   ```
2. 预期输出：`"function"`（如返回 `undefined`，说明 COOP/COEP 配置有问题）

### 步骤 4：功能测试

### 常见问题排查

#### 问题 1：SharedArrayBuffer 不可用
**现象**：Console 显示 `SharedArrayBuffer is not defined`
**解决**：
1. 确认使用 `npm run dev` 启动（vite.config.ts 已配置 COOP/COEP）
2. 不要使用 `file://` 协议直接打开 HTML
3. 检查浏览器是否禁用了 SharedArrayBuffer：
   - 地址栏输入 `chrome://flags/#shared-array-buffer`
   - 确保设置为「Default」或「Enabled」

#### 问题 2：ffmpeg.wasm 加载失败
**现象**：导出时卡住或报错
**解决**：
1. 检查网络连接，确保能访问 unpkg.com
2. 清空浏览器缓存后重试
3. 在 Console 查看具体错误信息

#### 问题 3：导出视频无画面
**现象**：导出的视频是黑屏
**解决**：
1. 确认导出前画布有可见内容
2. 检查模板背景色与笔迹颜色对比度
3. 尝试降低导出分辨率

#### 问题 4：导出速度慢
**现象**：1080p 导出超过 2 分钟
**解决**：
1. 关闭其他占用内存的标签页
2. 降低导出分辨率或质量
3. 使用高性能模式（插电使用）
